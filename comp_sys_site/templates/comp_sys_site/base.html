CompSysRankings

CompSysRankings is a metrics-based ranking of top computer systems institutions in the United States. Click on a plus (+) to expand an institution.

{% block content %}{% endblock %}

The publication data was last updated June 8, 2024. All publication data is pulled from the DBLP.

The source code for this website is here: https://github.com/j-mckerracher/comp_sys_rankings.

The backend code is available here: https://github.com/j-mckerracher/comp_sys_rankings_backend.

The author affiliation data comes from https://csrankings.org/.
